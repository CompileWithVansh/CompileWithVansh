<!--
═══════════════════════════════════════════════════════════════════
 SETUP — How to make everything work
═══════════════════════════════════════════════════════════════════

1. Create a new public repo with the EXACT name: CompileWithVansh
   (same as your username — that's what makes it your profile README)

2. Add this file as README.md in that repo

3. To enable the SNAKE animation:
   Create .github/workflows/snake.yml in the SAME repo with this content:

────────────────────────────────────────────────────────────────────
name: Generate Snake

on:
  schedule:
    - cron: "0 */24 * * *"
  workflow_dispatch:
  push:
    branches:
      - main

jobs:
  generate:
    runs-on: ubuntu-latest
    permissions:
      contents: write
    steps:
      - uses: Platane/snk/svg-only@v3
        with:
          github_user_name: CompileWithVansh
          outputs: |
            dist/github-contribution-grid-snake-dark.svg?palette=github-dark&color_snake=#EC4899&color_dots=#1a1f2e,#7c2d6f,#be185d,#ec4899,#f59e0b

      - uses: crazy-max/ghaction-github-pages@v3.1.0
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
────────────────────────────────────────────────────────────────────

4. After the workflow runs once (manually trigger it from Actions tab),
   the snake SVG will appear at:
   https://raw.githubusercontent.com/CompileWithVansh/CompileWithVansh/output/github-contribution-grid-snake-dark.svg

5. Profile views counter, stats, streak, top-langs, trophies, activity
   graph — all work automatically. No setup needed.
